# PERSONAL-FR-004: Email Drafting

## Requirement

The system must draft emails from user instructions or document context.

## Scope

- Professional reply
- Follow-up email
- Sales outreach draft
- Support reply draft
- Tone adjustment

## Acceptance Criteria

- Given a user provides email intent and context, when the assistant drafts an email, then the draft is editable and matches the requested tone.
- Given required context is missing, when drafting is requested, then the assistant asks for clarification.

## Out of Scope

- Autonomous sending
- Inbox management
