# PERSONAL-FR-002: Document Summary

## Requirement

The system must summarize uploaded documents into useful personal productivity outputs.

## Scope

- Short summary
- Key points
- Action items
- Questions to ask

## Acceptance Criteria

- Given a processed document, when the user requests a summary, then the system returns a concise summary and action items.
- Given the document is too unclear, when summarization runs, then the system explains the limitation.

## Out of Scope

- Legal interpretation
- Medical interpretation
- Financial advice
