# SALES-FR-004: Charts and Reports

## Requirement

The system must generate simple charts and report summaries from business data.

## Scope

- Bar chart
- Line chart
- Table summary
- Weekly report draft
- Email-ready summary

## Acceptance Criteria

- Given a user asks for a chart, when the data supports it, then the system generates a suitable chart.
- Given a weekly report is requested, when report generation completes, then the user receives a concise summary with key changes.

## Out of Scope

- Full BI dashboard builder
- Complex custom reports
