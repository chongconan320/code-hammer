# ECOM-FR-005: Sales Dashboard

## Requirement

The system should provide a basic dashboard for seller performance and support issues.

## Scope

- Orders
- Revenue
- Top products
- Repeat questions
- Common customer issues

## Acceptance Criteria

- Given sales/order data is available, when the seller opens the dashboard, then key metrics are displayed.
- Given customer messages are categorized, when dashboard data is shown, then common issues are summarized.

## Out of Scope

- Advanced analytics
- Ads optimization
- Inventory forecasting
